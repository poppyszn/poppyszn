<h2>🚀 What I Do</h2>

I design and manage AWS infrastructure with a strong focus on **security**, **cost efficiency**, and **automation**:

- 🏗️ Architected containerised workloads on **AWS EKS** with **ArgoCD**-driven GitOps deployments
- 🔐 Applied **security hardening** and incident response across EC2 and Kubernetes workloads, including forensic investigation and remediation
- 💸 Reduced AWS costs significantly through **Savings Plans**, instance rightsizing, and resource optimisation
- 🤖 Built end-to-end **CI/CD pipelines** with **GitHub Actions** for monolithic and microservices architectures (NestJS monorepo, Docker multi-stage builds)
- 📊 Rolled out full observability with **Grafana**, **Prometheus**, **Loki**, **Uptime Kuma**, and **Gotify** for metrics, logs, alerting, and uptime monitoring
- 🌐 Configured **AWS ALB**, **WAF**, **Route 53**, and **Kubernetes Ingress** for secure traffic management, DNS routing, and WebSocket support
- 🔑 Managed secrets securely with **AWS Secrets Manager** and **External Secrets Operator** via IRSA
- 💾 Automated **backup and restoration workflows**, eliminating manual overhead and improving recovery readiness
